A notebook and supporting functions for fetching and investigating
data about test cricket.

* Required Python libraries

Here are the Python libraries you'll need to import in order to run
the notebook:

- matplotlib (2.0.0)
- numpy (1.13.3)
- pandas (0.22.0)
- pystan (2.17.0.0)
- scipy (0.19.0)
